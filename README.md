🐾 Cat Facts & Doggo Pics API
A simple, fun web application that fetches random, mind-blowing facts about cats and pairs them with adorable, high-quality images of dogs. Because why choose one when you can have both?


✨ Features:-
Random Cat Facts: Fetches unique trivia about our feline friends using the Cat Fact API.

Infinite Doggo Gallery: Generates a new, random dog image every time you click using the Dog API.

Responsive Design: Works perfectly on desktops, tablets, and phones.

One-Click Refresh: Get a fresh combo of "Aww" and "Wait, really?" with a single button.


🚀 Live Demo: https://app.netlify.com/projects/dog-cat-api/
